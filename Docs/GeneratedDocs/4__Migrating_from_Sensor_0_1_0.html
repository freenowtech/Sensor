4. Migrating from Sensor 0.1.0

In order to use Sensor 0.2.0 you only need to import the SensorV2 framework. Sensor 0.1.0 is still available as the Sensor framework.

You should upgrade your feature to Sensor 0.2.0 if:

  • You want to benefit from the new features in Sensor 0.2.0, like the ability to cancel specific effects.
  • You discover a bug in Sensor 0.1.0 that affects you. Sensor 0.1.0 is now deprecated, so the bug will be fixed in Sensor 0.2.0. You'll need to upgrade your feature to Sensor 0.2.0 to benefit from the bugfix.
